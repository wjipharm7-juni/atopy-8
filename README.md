# 아토피 치료 전후 비교 랜딩페이지

황박사의 지혜로운 건강교실 - 유튜브 광고용 랜딩페이지

## 프로젝트 개요

아토피 치료의 실제 성공 사례를 보여주는 전문 랜딩페이지입니다. 치료 전후 비교 사진과 실제 환자 가족들의 감사 메시지를 통해 치료 효과를 시각적으로 전달합니다.

## 주요 기능

### ✅ 완료된 기능

1. **반응형 디자인**
   - 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 레이아웃
   - 깔끔하고 현대적인 그라데이션 디자인

2. **성인 남성 치료 사례 섹션**
   - 4개 부위 치료 전후 비교 (얼굴, 가슴, 다리, 팔꿈치)
   - 각 사진에 명확한 '치료 전/후' 라벨링
   - 화살표로 변화 흐름 시각화

3. **초등 여학생 치료 사례 섹션**
   - 3개 부위 치료 전후 비교 (팔 2곳, 무릎)
   - 완치 감사 문자 메시지 포함
   - 별도 배경색으로 섹션 구분

4. **여자 아기 건강 근황 섹션**
   - 북경에서 치료받은 아기의 엄마가 보낸 4개의 이메일
   - 그리드 레이아웃으로 깔끔하게 정리
   - 치료 후 건강한 근황과 감사 메시지

5. **연락처 정보**
   - 상담 전화: 031-439-6557
   - 상담 시간: 10:00 - 18:00
   - 아이콘으로 시각적 강조

6. **세련된 UI/UX**
   - 그라데이션 컬러 스킴 (보라-파랑 계열)
   - 호버 효과로 인터랙티브한 경험
   - 부드러운 그림자와 둥근 모서리
   - 현대적인 타이포그래피

1. **헤더**
   - 황박사의 지혜로운 건강교실 타이틀
   - 아토피 치료의 새로운 희망 태그라인

2. **소개 섹션**
   - 완치된 분들의 놀라운 변화 소개

3. **성인 남성 사례**
   - 4개의 비교 카드 세트
   - 각 부위별 치료 전후 이미지

4. **초등 여학생 사례**
   - 3개의 비교 카드 세트
   - 완치 감사 문자 메시지 박스

5. **여자 아기 근황**
   - 4개의 이메일 카드
   - 건강한 근황 업데이트

6. **연락처**
   - 전화번호 및 상담 시간 정보

7. **푸터**
   - 저작권 정보

## 디자인 특징

### 색상 팔레트
- **주요 그라데이션**: #667eea → #764ba2 (보라-파랑)
- **배경**: #f5f7fa → #c3cfe2 (밝은 그레이-블루)
- **치료 전 라벨**: #f093fb → #f5576c (핑크-레드)
- **치료 후 라벨**: #4facfe → #00f2fe (블루-시안)

### 타이포그래피
- **폰트**: Noto Sans KR (Google Fonts)
- **가독성**: 다양한 폰트 웨이트로 정보 계층 구조화

### 레이아웃
- **카드 디자인**: 흰색 배경, 그림자, 둥근 모서리
- **반응형 그리드**: 화면 크기에 따라 자동 조정
- **호버 효과**: 카드에 마우스 올리면 살짝 상승

## 특이사항

1. **용어 사용**
   - "환자" 대신 "성인 남성", "초등 여학생", "여자 아기"로 표현
   - 친근하고 부드러운 톤 유지

2. **이미지 처리**
   - 모든 이미지는 외부 URL에서 로드
   - 각 이미지는 400px 고정 높이로 통일
   - object-fit: cover로 비율 유지

3. **단일 페이지 구성**
   - 모든 내용이 한 페이지에 포함
   - 링크나 상담 폼 없음 (요청사항에 따름)
   - 스크롤로 모든 내용 확인 가능
